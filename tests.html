  • Application filtering restricts user's ability to access certain applications, wherever they may live, based on Layer 7 attributes.

  • This test attempts to load the following URL: https://bittorrent.com

  • Application filtering allows the administrator to block specific traffic based on Layer 7 application attributes. In this test, we are testing your browser's connectivity to https://bittorrent.com

  • Application filtering requires Next Generation Firewall (NGFW) technology. Appliances with NGFW capabilities will have built in app-id engines, that typically consider things like destination IP, port/protocol, initial handshake behaviors, and packet size to match against a list of predefined applications and their known behaviors. Most NGFW come with thousands of predefined applications, but also support the ability to custom define app-match criteria for homegrown applications.

Deep Inspection Tests


  • This test attempts to download an EICAR test file over a TLS encrypted connection (https).

  • This file is a test file only, and will not harm your computer

  • File can be downloaded from the following URL: https://secure.eicar.org/eicar.com

  • In this test, we are attempting to download an EICAR test file over a TLS/SSL encrypted session. Most modern web traffic is TLS encrypted, meaning you will need a solution which can handle SSL decryption in order to stop the downloading of malicious content over https. We are using eicar.org/eicar.com to test.

  • Without SSL decryption, users are able to download any files over https as that session would otherwise be encrypted end to end.

  • SSL decryption requires a firewall to decrypt user's sessions, inspect the payload, and re encrypt the session before passing to the client/server. To do this, a trusted certficate must be installed on both the firewall and the host machine.

  • Some solutions are starting to support things like TLS fingerprinting, which enforces policy based on TCP and TLS handshake behaviors. This eliminates the need for certificate management, though the overall security efficacy may suffer as a result
